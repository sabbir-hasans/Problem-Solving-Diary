# 🔢 Variables

## 📌 What is a Variable?

**Variable** হলো data store করার একটি container।

👉 সহজভাবে:
> Variable = একটা নাম দেওয়া box, যেখানে data রাখা হয়

---

## 💡 Example (C)

```c
#include <stdio.h>

int main() {
    int a = 10;
    float b = 5.5;
    char c = 'A';

    printf("%d %f %c", a, b, c);
    return 0;
}
