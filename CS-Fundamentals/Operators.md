# ➕ Operators

## 📌 What is an Operator?

**Operator** হলো এমন symbol যা দিয়ে calculation বা comparison করা হয়।

👉 সহজভাবে:
> Operator = operation করার tool

---

## 🔢 Types of Operators

### 1️⃣ Arithmetic Operators

| Operator | Meaning        | Example      |
|----------|---------------|-------------|
| +        | Addition      | a + b       |
| -        | Subtraction   | a - b       |
| *        | Multiplication| a * b       |
| /        | Division      | a / b       |
| %        | Modulus       | a % b       |

---

## 💡 Example (Arithmetic)

```c
int a = 10, b = 3;

printf("%d\n", a + b); // 13
printf("%d\n", a - b); // 7
printf("%d\n", a * b); // 30
printf("%d\n", a / b); // 3
printf("%d\n", a % b); // 1
